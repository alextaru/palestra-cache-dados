# PALESTRA CACHE DADOS

## Requerimentos
* [NodeJs](https://nodejs.org/en/)
* [Docker](https://www.docker.com/)
* [Docker-compose](https://docs.docker.com/compose/install/)

## Instalação
```bash
$ npm install
```
```bash
$ npm run docker
```
```bash
$ npm run database
```

## Executar
```bash
$ npm start
```
